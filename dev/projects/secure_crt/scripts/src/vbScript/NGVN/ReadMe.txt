The Scripts/SecureCRT/NGVN directory contains scripts that directly pertain to NGVN.  These may contain SecureCRT specific scripts or any other scripts specific to NGVN.

Required Files:
	Constants.txt is used by the scripts in this folder to set global constant variables.
	BootStrap.txt is used by the NGVN scripts to load the Constants.txt and load any needed class files to support the scripts.
